## Temporary Projects

### This extension adds a `File > New > Temporary Project...` menu item

![image](https://user-images.githubusercontent.com/11719160/42604757-51550320-856c-11e8-96b2-76513c1c7159.png)

### Alternatively you can use `Create new > temporary project...` on the `Start Page`

![image](https://user-images.githubusercontent.com/11719160/46201343-c57a8e00-c30b-11e8-96c8-60d1b03b716d.png)

### Clicking it will open the regular New Project dialog

![image](https://user-images.githubusercontent.com/11719160/42604832-a4fd8092-856c-11e8-8ebd-90fe5c0a5f4b.png)

The default project location will have `Temp\yyyy-MM-dd\` appended to it. This will hopefully help organize temporary projects and avoid ending up with `ClassLibrary17`. ðŸ˜‰
